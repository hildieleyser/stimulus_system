# Stimulus Generators

Tools for creating high-quality naturalistic stimuli for neural decoding experiments.

## Overview

This package provides:
- **Procedural abstract stimulus generation** (Tier 1)
- **Dataset integration guidance** (Tiers 2-4)
- **Validation and quality control**

## Quick Start

### Generate Tier 1 Abstract Stimuli

```bash
# Generate all Tier 1 stimuli (Gabor patches, shapes, tones)
python setup_stimuli.py --tiers 1 --categories dog,car,apple,chair

# Visual stimuli only
python generate_abstract_visual.py --categories dog,car,apple,chair --n-per-category 10

# Audio stimuli only
python generate_abstract_audio.py --categories dog,car,apple,chair --n-per-category 9
```

### Show Instructions for Higher Tiers

```bash
# Display instructions for Tiers 2-4
python setup_stimuli.py --tiers 2,3,4 --instructions-only
```

### Validate Stimulus Setup

```bash
# Validate directory structure
python setup_stimuli.py --validate-only

# Full system validation
cd ..
python run_experiment.py --dry-run
```

## Stimulus Hierarchy

### Tier 1: Abstract (Procedurally Generated)

**Visual:**
- Gabor patches (oriented sinusoidal gratings)
- Geometric shapes (circles, squares, triangles, diamonds)
- Grayscale, minimal semantic content
- Category-specific parameters for reproducibility

**Auditory:**
- Harmonic complex tones (fundamental + harmonics)
- Band-limited noise bursts
- Amplitude-modulated (AM) tones
- Category-specific parameters for reproducibility

**Generated by:** `generate_abstract_visual.py`, `generate_abstract_audio.py`

### Tier 2: Single Objects & Environmental Sounds

**Visual:**
- Isolated objects on neutral backgrounds
- Clear category identity
- Source: ImageNet, COCO, or custom photography

**Auditory:**
- Environmental sounds (dog barks, car engines, etc.)
- 1-2 second clips
- Source: ESC-50, FSD50K, Freesound

**Integration:** Manual download and organization (see instructions)

### Tier 3: Objects in Context & Spoken Words

**Visual:**
- Objects in simple scene contexts
- Category object is clear focal point
- Source: COCO, Places365, photography

**Auditory:**
- Spoken category labels ("dog", "car", etc.)
- Auto-generated via TTS or recorded
- Can be generated by system (see config.yaml)

**Integration:** Semi-automated (TTS) or manual

### Tier 4: Naturalistic Scenes & Sentences

**Visual:**
- Complex naturalistic scenes
- Rich perceptual detail
- Source: COCO, Places365, photography

**Auditory:**
- Descriptive sentences about category
- Natural prosody and intonation
- Can be generated via TTS

**Integration:** Semi-automated (TTS) or manual

## Dataset Sources

### Visual

- **ImageNet**: https://www.image-net.org/
  - Massive image database organized by concepts
  - Download API available
  - Suggested for Tier 2

- **COCO**: https://cocodataset.org/
  - 330K images with object annotations
  - Good for Tiers 2-4
  - Easy to filter by object categories

- **Places365**: http://places2.csail.mit.edu/
  - Scene-centric dataset
  - Suggested for Tier 4

### Auditory

- **ESC-50**: https://github.com/karolpiczak/ESC-50
  - 2000 environmental sounds in 50 classes
  - Pre-segmented 5-second clips
  - Suggested for Tier 2

- **FSD50K**: https://zenodo.org/record/4060432
  - 50,000+ annotated sound clips
  - Wide variety of categories
  - Alternative for Tier 2

- **Freesound**: https://freesound.org/
  - Community sound database
  - Creative Commons licensed
  - Good for filling gaps

### Speech (TTS)

- **gTTS** (Google Text-to-Speech): Auto-generated by system
- **Common Voice**: https://commonvoice.mozilla.org/
- **LibriSpeech**: http://www.openslr.org/12/

## Scripts Reference

### setup_stimuli.py

Master orchestration script.

```bash
python setup_stimuli.py --tiers 1,2,3,4 --categories dog,car,apple,chair
```

**Options:**
- `--tiers`: Which tiers to setup (comma-separated)
- `--categories`: Category list (comma-separated)
- `--output-dir`: Output directory (default: ../stimuli/)
- `--n-stimuli`: Number per category (default: 10)
- `--instructions-only`: Show instructions without generating
- `--validate-only`: Validate existing stimuli

### generate_abstract_visual.py

Generate Tier 1 visual stimuli.

```bash
python generate_abstract_visual.py --categories dog,car --n-per-category 10 --type both
```

**Options:**
- `--categories`: Category list
- `--output-dir`: Output directory
- `--n-per-category`: Number of stimuli per category
- `--type`: `gabor`, `shapes`, or `both`

### generate_abstract_audio.py

Generate Tier 1 audio stimuli.

```bash
python generate_abstract_audio.py --categories dog,car --n-per-category 9 --duration 0.5
```

**Options:**
- `--categories`: Category list
- `--output-dir`: Output directory
- `--n-per-category`: Number of stimuli per category
- `--duration`: Duration in seconds
- `--sample-rate`: Sample rate (Hz)
- `--types`: Comma-separated tone types

## Quality Guidelines

### Visual Stimuli

- **Resolution**: 400x400px minimum
- **Format**: PNG (preferred) or JPG
- **Color**: Grayscale for Tier 1, RGB for higher tiers
- **Background**: Neutral/consistent within tier

### Audio Stimuli

- **Sample rate**: 44.1kHz recommended
- **Bit depth**: 16-bit or 32-bit float
- **Format**: WAV (uncompressed)
- **Loudness**: System will normalize to -23 LUFS
- **Duration**:
  - Tier 1: 0.5s
  - Tier 2: 1-2s
  - Tier 3: 0.5-1s (single word)
  - Tier 4: 2-4s (sentence)

## Troubleshooting

### "ModuleNotFoundError"

Install required packages:
```bash
pip install numpy pillow soundfile scipy
```

### "Directory not found"

Ensure you're running from stimulus_generators/ directory or use absolute paths:
```bash
cd /Users/hildieleyser/stimulus_system/stimulus_generators
python setup_stimuli.py --tiers 1
```

### "No stimuli found in dry-run"

1. Check directory structure matches expected format
2. Run validation: `python setup_stimuli.py --validate-only`
3. Ensure file naming follows convention: `<category>_<tier>_<N>.png/wav`
